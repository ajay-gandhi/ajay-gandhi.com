Projects

Sage

Sage is a vocal personal assistant built on Node.js. It always listens, and answers almost any question rapidly. Sage can read you your unread email, tell you whether it will rain, and give you a rundown on Socrates.

It implements a straightforward module system that allows you to enable and disable any feature, or add your own. Sage is open source, so feel free to browse the code and contribute.

Ezra

Intended to replace PeopleSoft's Student Center from the 1800s, Ezra is a native desktop application that greatly simplifies interacting with Cornell's student services. It crawls Cornell's various and (unfortunately) distributed websites to consolidate any relevant information into a clean interface.

You can only use Ezra if you're a Cornell student, since you must login with your Cornell NetID and password. However, PeopleSoft Student Center is actually a widely used student management system, so it shouldn't be too difficult to make a few tweaks and use Ezra elsewhere.

The application is hosted on GitHub, so you can see the code there.

Facebook CLI

Formed as a hackathon project at BigRedHacks 2014, Facebook CLI is an application designed to help you procrastinate anytime you're at work or coding otherwise. Using a headless browser, the app allows you to access your Facebook newsfeed directly from the command line, without opening any other applications. You can also like and comment on posts and poke your friends.

Check it out on GitHub.

Facebook Autopoker

By combining request and cheerio, I created a basic headless browser. Users submit a really short form, which sends their login information to the server. The server takes care of the rest.

The Autopoker is online, so get started at http://poke-aa7.rhcloud.com! It's a very straightforward codebase, and it's on GitHub.

Congo

Congo is a texting utility. Try texting any question whatsoever to +1 (262) 622-6646 and watch it respond with the correct answer. A few good examples are:

  • How many words are in the English language?
  • How tall is Michael Jordan?
  • What is 84 divided by 12?

Congo is not yet open-source, but it may be soon!

NPM Packages

I've made a few contributions to NPM's package repository. These include PRs to projects such as xkeychain as well as packages of my own. Here's a short list:

  • cli-pipe: Accept piped data as an argument to your CLI app
  • find-synonyms: Find synonyms for words
  • hs: Store and load large pieces of data publicly
  • LightTTS: A lightweight text to speech module
  • NPMaybe: Test NPM modules without installing them
  • Speakable: Speech to text module for Node.js
  • svme: Store and load temporary data with your terminal
  • textbelt: Send unlimited free texts from Node.js
  • xkeychain: Securely store credentials in the native OS keychain

Facebook Photo Finder

This app allows you to search through your photos by keyword. In other words, find pictures of you that are "amazing" or "gorgeous". The app creates a list of synonyms for your keyword, then searches through each image's comments and description for any of these synonyms.

The app is online, so try it out at http://photo-aa7.rhcloud.com! It's a very straightforward codebase, and it's on GitHub.

Asciify Image

I mostly created this module to learn more about the image asciification process (it's pretty common, but clever). In the end, I added some interesting and useful features, such as a command-line interface for asciifying images. The project can also be used as a Node.js module.

Learn more or see the code.

Bomper

This command line tool allows you to open your web bookmarks directly from the command line. The interface is really simple and easy to use, and it crawls bookmarks from all of your browsers.

See the repo for a list of supported browsers and installation instructions.

Ultimate TermTacToe

A simple game of tic tac toe could kill a few seconds of your time. But a game of ultimate tic tac toe? The strategies are endless. If you're not familiar with ultimate tic tac toe, you can read the rules. This version runs in your command line. See the readme in the repo for more info!
